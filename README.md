# ĐỒ ÁN FRONT-END CƠ BẢN - PETSPATIAL

Đồ án Pespatial là một trong 2 đồ án lớn tốt nghiệp khóa học FEB19 Kmin đối với dự án ReactJS. Nó là đồ án viết về template dành cho các trung tâm chăm sóc thú cưng hoặc các cá nhân, các bạn có shop thú cưng muốn giới thiệu shop của mình đến với mọi người.

## Quick start

Quick start options:
- Clone the repo: `git clone https://github.com/duongnx97/Petspatial.git`.
- `npm i`
- [Download from Github](https://github.com/Minh306/cybersoft-movie).
